## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Pattarapon0/merge_sort_unit_test.git
   cd merge_sort_unit_test
   ```

2. Install dependencies:
   ```bash
   npm i
   ```
## Testing

Run the tests using:

```bash
npm test
```

This will execute the test suite using uvu and tsx.
